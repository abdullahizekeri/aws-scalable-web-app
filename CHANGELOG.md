# Changelog

All notable changes to this project will be documented here.

## [1.0.0] - 2026-06-14
### Added
- Initial project setup
- VPC with public and private subnets across 2 AZs
- NAT Gateways for private subnet internet access
- EC2 Launch Template with userdata script
- Application Load Balancer with HTTPS listener
- Auto Scaling Group with target tracking policy
- WAF with OWASP managed rules
- RDS MySQL Multi-AZ instance
- CloudFront distribution
- Route 53 alias record
- CloudWatch dashboards and alarms
- SNS alerts for critical metrics
- Systems Manager Session Manager access
